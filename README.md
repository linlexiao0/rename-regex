# rename-regex
A Command Line Tool for Rename Files by Regex

## run
```bash
python rere.py
```
or (Linux only)
```bash
chmod +x rere.py
./rere
```
or (Windows only)
```cmd
rere
```

## usage
`usage: rere.py [-h] [-f] [-r] src dest`
```
  -h, --help       show this help message and exit
  -f, --force      never prompt before rename
  -r, --recursive  rename files in subdirs
```
